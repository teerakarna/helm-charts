ScoutSuite is scheduled to audit {{ .Values.provider | upper }}.

Workload: {{ .Values.workloadType | upper }}
{{- if eq .Values.workloadType "cronjob" }}
Schedule: {{ .Values.schedule }}

Trigger a scan immediately:
  kubectl create job --from=cronjob/{{ include "scoutsuite.fullname" . }} \
    {{ include "scoutsuite.fullname" . }}-manual \
    -n {{ .Release.Namespace }}
{{- end }}

Check scan status:
  kubectl get jobs -n {{ .Release.Namespace }} -l app.kubernetes.io/instance={{ .Release.Name }}

View scan logs:
  kubectl logs -n {{ .Release.Namespace }} \
    -l app.kubernetes.io/instance={{ .Release.Name }} --tail=50

{{- if .Values.persistence.enabled }}

View report (port-forward a pod with access to the PVC, or mount the PVC in a debug pod):
  helm install debug teerakarna/sleep -n {{ .Release.Namespace }} \
    --set persistence.enabled=true \
    --set 'volumes[0].name=reports' \
    --set 'volumes[0].persistentVolumeClaim.claimName={{ include "scoutsuite.pvcName" . }}'
  kubectl exec -it -n {{ .Release.Namespace }} \
    $(kubectl get pod -n {{ .Release.Namespace }} -l app.kubernetes.io/instance=debug -o jsonpath="{.items[0].metadata.name}") \
    -- sh
  # Inside the pod: ls /reports
{{- end }}

Delete when done (Job mode only):
  helm uninstall {{ .Release.Name }} --namespace {{ .Release.Namespace }}
